HTML CHEATSHEET

Tag Description
<!--...--> This defines a comment
<!DOCTYPE> Defines the document type
<a> Defines a hyperlink
<body> Defines the document body
<br> Defines a single line break
<button> Defines a clickable button
<div> Defines a section in a document
<em> Defines an emphasized text
<footer> Defines a footer for a document
<form> Defines a HTML form for user input
<h1> to <h6> Defines HTML headings
<head> Contains metadata/information for the document
<header> Defines a header for a document or section
<html> Defines the root of a HTML document
<i> Usually used to define an icon element
<img> Defines an image
<input> Defines an input control
<label> Defines a label for an input element
<li> Defines a list item
<link> Defines the relationship between the document and an external resource
<nav> Defines navigation links
<ol> Defines an ordered list
<p> Defines paragraph
<script> Defines a client-side script
<span> Defines an inline document section
<style> Defines style information for a document
<table> Defines a table
<tbody> Groups the body content in a table
<td> Defines a cell in a table
<textarea> Defines a multiline input control
<th> Defines a header cell in a table
<thead> Groups the header content in a table
<tr> Defines a row in a table
<ul> Defines an unordered list

Attribute Description
alt Specifies an alternate text when the original element fails to display
class Specifies one or more classnames for an element
cols Specifies the visible width of a text area
for Specifies which form element(s) a label/calculation is bound to
href Specifies the URL of the page the link goes to
id Specifies the URL of the page the link goes to
method Specifies the HTTP method to use when sending form-data
name Specifies the name of the element
onclick Script to be run when the element is being clicked
rel Specifies the relationship between the current document and the linked document
rows Specifies the visible number of lines in a text area
src Specifies the URL of the media file
type Specifies the type of element
value Specifies the value of the element
